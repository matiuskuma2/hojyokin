## 3. jGrants API - 公募要領添付(has_data=true)だがURL取得不可
## 対象: 1,215件
→ jGrants公開APIではPDF URLが提供されない
→ ポータルページ(SPA)のスクレイピングが必要(Playwright等)
→ 各案件の https://www.jgrants-portal.go.jp/subsidy/{id} ページからダウンロード

## 4. 再探索ルール
1. 手動登録案件(200_no_pdf): 出典URLの下層ページ(1階層下)を探索
2. 手動登録案件(404): ドメインルートを辿り、類似パスを検索
3. 情報の泉(dead_link): detail_url(j-izumi.com)を起点に、自治体公式ページの最新URLを取得
4. jGrants(has_data=true): PlaywrightでポータルページのダウンロードボタンをクリックしてPDF取得
